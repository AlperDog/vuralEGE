# Vural EGE - Haber Sitesi

Modern, responsive ve estetik bir haber sitesi. Fayn.press benzeri tasarım ile Greek Island 1064A rengi tonlarını kullanan, mavi-beyaz ağırlıklı, Helvetica fontlarına sahip ve siyah süslemeli bir haber platformu.

## Özellikler

- ✅ **Responsive Tasarım**: Tüm cihazlarda mükemmel görünüm
- ✅ **Modern UI/UX**: Temiz ve kullanıcı dostu arayüz
- ✅ **Renk Paleti**: Greek Island 1064A tonlarında mavi-beyaz ağırlıklı tasarım
- ✅ **Tipografi**: Helvetica font ailesi
- ✅ **Siyah Süslemeler**: Modern ve şık siyah vurgular
- ✅ **Performans**: Hızlı yükleme ve optimize edilmiş kod

## Teknolojiler

- HTML5
- CSS3 (Grid, Flexbox)
- Vanilla JavaScript
- Responsive Design

## Kurulum

1. Projeyi klonlayın:
```bash
git clone https://github.com/AlperDog/vuralEGE.git
```

2. Proje dizinine gidin:
```bash
cd vuralEGE
```

3. `index.html` dosyasını bir web tarayıcısında açın.

## Yapı

```
vuralEGE/
├── index.html      # Ana HTML dosyası
├── styles.css      # Stil dosyası
├── script.js       # JavaScript dosyası
└── README.md       # Bu dosya
```

## Özelleştirme

### Renkleri Değiştirme

`styles.css` dosyasındaki `:root` değişkenlerini düzenleyerek renkleri özelleştirebilirsiniz:

```css
:root {
    --primary-blue: #1e3a8a;
    --secondary-blue: #2563eb;
    /* ... diğer renkler */
}
```

### İçerik Ekleme

`index.html` dosyasındaki haber kartlarını ve içerikleri düzenleyerek kendi içeriklerinizi ekleyebilirsiniz.
